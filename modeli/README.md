# modeli / Models

Ovaj folder sadrži trenirane modele i povezane fajlove.

## Broj fajlova: 1 (plus podfolderi)
- `registry.yaml`
- Podfolderi: `moj-bot`, `mlruns`

## Opis fajlova i podfoldera

1. **registry.yaml**  
   - **Što radi?** Registar modela – popis svih treniranih modela i njihovih verzija.  
   - **Primjer upotrebe**: Praćenje verzija modela s MLflow-om.

2. **moj-bot**  
   - **Što radi?** Sadrži tvoj lokalni trenirani model.  
   - **Primjer upotrebe**: Pohrana težina tvog modela.

3. **mlruns**  
   - **Što radi?** Sadrži MLflow praćenje (logovi treniranja, metrike).  
   - **Primjer upotrebe**: Pregled performansi modela.
