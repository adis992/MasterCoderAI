# evaluacija / Evaluation

Ovaj folder sadrži kod za testiranje performansi modela.

## Broj fajlova: 0 (samo podfolderi)
- Podfolderi: `benchmark`, `testovi`

## Opis podfoldera

1. **benchmark**  
   - **Što radi?** Usporedni testovi performansi modela.  
   - **Primjer upotrebe**: Uspoređuje tvoj model s drugim modelima.

2. **testovi**  
   - **Što radi?** Scenario-based testiranje modela.  
   - **Primjer upotrebe**: Testira model na programerskim scenarijima.
